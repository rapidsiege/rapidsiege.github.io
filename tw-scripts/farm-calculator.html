🚀 First Setup

1 Connect the Village Database

Go to the Village Database tab → click Connect DB Folder → select the folder containing village.txt and player.txt (the same folder as this HTML file).

The folder is remembered for future sessions. If the files change, click Re-sync Files to reload them.

2 Set your Server URL

In the top bar, enter your server URL — e.g. es100.guerrastribales.es. This is used to build the attack links in the Calculator.

3 Configure World Settings

Set World Speed and Unit Speed to match your server. For es100: World Speed 2×, Unit Speed 0.5×.

4 Import your villages

In the My Villages tab, drag the gold bookmarklet button to your browser's bookmarks bar. Then open your TW account → Overview → Units tab and click the bookmark — a JSON file will download.

Come back here and click Import My Villages to load it. Your villages, coordinates and troop counts are imported automatically.

5 Add target villages

Go to Target Villages → click + Add Target. Enter the X and Y coordinates first — the database will auto-fill the name, village ID and player.

Set building levels (Sawmill, Clay Pit, Iron Mine), Warehouse level and optionally Hiding Place level. Paste Last Loot data if you have it — this gives much more precise estimates.

🚀 Configuración Inicial

1 Conectar la base de datos

Ve a la pestaña Base de Datos → haz clic en Conectar Carpeta BD → selecciona la carpeta que contiene village.txt y player.txt (la misma carpeta donde está este archivo HTML).

La carpeta se recuerda para futuras sesiones. Si los archivos cambian, haz clic en Resincronizar archivos para recargarlos.

2 Configurar la URL del servidor

En la barra superior, introduce la URL de tu servidor — ej. es100.guerrastribales.es. Se usa para generar los enlaces de ataque en la Calculadora.

3 Configurar la velocidad del mundo

Ajusta Velocidad del Mundo y Velocidad de Unidades según tu servidor. Para es100: Velocidad 2×, Velocidad de Unidades 0.5×.

4 Importar tus pueblos

En la pestaña Mis Pueblos, arrastra el botón dorado a la barra de marcadores de tu navegador. Luego abre tu cuenta en GT → Visión General → Tropas y haz clic en el marcador — se descargará un JSON.

Vuelve aquí y haz clic en Importar Mis Pueblos. Tus pueblos, coordenadas y tropas se importan automáticamente.

5 Añadir pueblos objetivo

Ve a Pueblos Objetivo → haz clic en + Añadir Objetivo. Introduce primero las coordenadas X e Y — la base de datos rellenará automáticamente el nombre, ID de pueblo y jugador.

Configura los niveles de edificios (Leñador, Barrera, Mina de Hierro), nivel de Almacén y, opcionalmente, nivel de Escondrijo. Pega los datos del último saqueo si los tienes — mejora mucho la precisión de las estimaciones.

⚔ Daily Use

1 Update your troops

Re-run the bookmarklet on your TW Overview → Units page and click Import My Villages again. It updates existing villages in place — no duplicates.

Or go to My Villages and edit any village manually to adjust LC/HC counts.

2 Log a raid result

After attacking, go to Target Villages → click Edit on the target. Paste the leftover resources (e.g. 260 238 81) into Last Loot Left and the timestamp (e.g. 25.05.26 20:13:25:000) into Last Loot Time.

The calculator will project resources forward from that point, accounting for production at each building level.

3 Use the Calculator

Go to Calculator → select a target from the dropdown. Your villages are listed sorted by distance.

Attack buttons are bright when you have enough troops for that unit type, and dimmed when you don't. Villages with zero troops are hidden. Click ⚔ LC or ⚔ HC to open the rally point pre-filled.

4 Re-sync the database

If you download updated map files (village.txt / player.txt), go to Village Database → click Re-sync Files to reload the latest data.

Use the search bar to find any village by name, coordinates or player, and add it as a target with one click.

5 Save & backup

Data auto-saves to a local file (tw_farm_data.json) and to browser storage as a backup. Use Export JSON in the top bar to download a manual backup, or Import JSON to restore one.

⚔ Uso Diario

1 Actualizar tus tropas

Vuelve a ejecutar el marcador en tu página de GT Visión General → Tropas y haz clic en Importar Mis Pueblos de nuevo. Actualiza los pueblos existentes sin duplicarlos.

O ve a Mis Pueblos y edita cualquier pueblo manualmente para ajustar los recuentos de Lijas/Pesadas.

2 Registrar el resultado de un ataque

Tras atacar, ve a Pueblos Objetivo → haz clic en Editar en el objetivo. Pega los recursos restantes (ej. 260 238 81) en Recursos Restantes y la marca de tiempo (ej. 25.05.26 20:13:25:000) en Hora del Último Saqueo.

La calculadora proyectará los recursos desde ese momento, teniendo en cuenta la producción según los niveles de edificios.

3 Usar la Calculadora

Ve a Calculadora → selecciona un objetivo en el desplegable. Tus pueblos aparecen ordenados por distancia.

Los botones de ataque están brillantes cuando tienes suficientes tropas de ese tipo, y apagados cuando no. Los pueblos sin tropas están ocultos. Haz clic en ⚔ Lijas o ⚔ Pesadas para abrir la plaza de reuniones pre-rellenada.

4 Resincronizar la base de datos

Si descargas archivos de mapa actualizados (village.txt / player.txt), ve a Base de Datos → haz clic en Resincronizar archivos para cargar los datos más recientes.

Usa la barra de búsqueda para encontrar cualquier pueblo por nombre, coordenadas o jugador, y añádelo como objetivo con un clic.

5 Guardar y hacer copia de seguridad

Los datos se guardan automáticamente en un archivo local (tw_farm_data.json) y en el almacenamiento del navegador como copia de seguridad. Usa Exportar JSON en la barra superior para una copia manual, o Importar JSON para restaurar una.
