Manage Defense

Run the Village Supports Exporter userscript (villageSupports.js) on your allied villages, then paste its export here or load the downloaded file(s). You can select several batch files at once — they are merged (a village in two files keeps the last file's data). It lists the support currently stationed in each village, broken down by origin.
Run the Target Village Orders Exporter userscript (incomingOrders.js) on your allied villages, then paste its export here or load the file(s). You can select several batch files at once — they are merged (a village in two files keeps the last file's orders). Only SUPPORT movements are imported — attacks are ignored. Without this, incoming support is estimated from the tribe export's inbound-troops data.

Server

Used to build rally-point links in the plan table. The UTC offset converts your local clock to server time for the launch-feasibility check (guerrastribales.es runs on UTC+2 in summer, UTC+1 in winter).

Backup & Debug

Exports everything saved locally — offensive targets, the generated plan, settings, language, the loaded troop data and the morale inputs for it — as one JSON file. Share it so a tribemate's exact situation can be reproduced for debugging. Importing REPLACES your current saved targets, plan, settings and map preferences.
